# 🛒 SuperMarket Management System

A Java-based SuperMarket Management System developed using Object-Oriented Programming (OOP) concepts. This project is designed to automate and simplify supermarket operations such as product management, seller management, billing, and user authentication.

The system provides a user-friendly interface where administrators can manage supermarket data and sellers can perform selling operations efficiently.

---

# 📌 Project Overview

The SuperMarket Management System focuses on managing daily supermarket activities digitally instead of using manual record keeping.

The application helps in:

- Managing products
- Managing sellers
- Handling sales records
- Generating bills
- Maintaining product quantities
- Providing secure login access

The project is developed in Java using Object-Oriented Programming principles to create a structured, reusable, and maintainable system.

---

# 🚀 Features

## 🔐 Login System

The system provides authentication for different users:

- Admin Login
- Seller Login

Users must enter valid credentials to access their respective modules.

---

## 👨‍💼 Admin Module

Admin has complete control over supermarket management.

Features:

- Connect with database
- Load product data
- Load seller data
- Add new products
- Delete products
- Add selling records
- Delete selling records
- Manage supermarket operations

---

## 📦 Product Management

This module manages all product-related operations.

Features:

- Add products
- Delete products
- Update product information
- Manage:

  - Product ID
  - Product Name
  - Quantity
  - Price
  - Category

---

## 👥 Seller Management

Seller information can be managed through this module.

Features:

- Add seller details
- Manage seller records
- Store:

  - Seller ID
  - Name
  - Password
  - Gender

---

## 🛍 Selling Module

This module handles customer purchases and billing.

Features:

- Load selling data
- Update product quantity after sale
- Search products
- Get product category
- Insert billing data
- Filter selling records

---

# 🧠 OOP Concepts Used

This project demonstrates important Object-Oriented Programming concepts:

## Classes and Objects

Different system entities are created as classes:

- Admin
- Login
- Product Manager
- Seller Manager
- Selling

Each class contains its own data and functions.

---

## Encapsulation

Data is protected inside classes and accessed using methods like:

- Getter methods
- Setter methods

---

## Inheritance

Classes reuse common functionality to reduce code duplication.

---

## Abstraction

Interfaces are used to hide unnecessary implementation details and provide a simple structure.

---

## Polymorphism

Multiple classes can implement the same interface and perform different operations.
